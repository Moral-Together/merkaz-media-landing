# WebsiteClaude02 — Merkaz Media

Одностраничный рекламный лендинг (иврит, RTL). Цель — заявки с рекламы:
приоритет №1 — разработка приложений, №2 — Web-системы и бизнес-сайты.

**Отличие от WebsiteClaude01:** ширина контента увеличена с 1200px до **1400px**
(переменная `--container` в `styles.css`). Вся раскладка адаптивная, поэтому на
экранах уже 1400px секции просто занимают больше ширины; точки перелома
(1080 / 768 / 480px) не изменились.

## Файлы

- `index.html` — вся страница (один лендинг, без подстраниц)
- `styles.css` — стили; палитра взята из логотипа (синий/маджента/зелёный/жёлтый + коралловая CTA)
- `script.js` — мобильное меню, reveal-анимации, аккордеон FAQ, обработка формы
- `assets/` — только реальные материалы из «Projects Main» (скриншоты VaadBot, M1 Radio, Moral for Good, MoralTogether, фото Владимира, логотипы клиентов)

## Перед запуском обязательно

1. **Форма** (`script.js`, блок `data-contact-form`) — сейчас показывает success-сообщение
   на клиенте. Подключить отправку: email / CRM / Make / webhook.
2. **Телефон и WhatsApp** — на странице их нет, т.к. номер не был известен.
   Рекомендую добавить кнопку WhatsApp рядом с CTA в секции контактов.
3. **Аналитика** — добавить GA4 / Google Ads тег + событие на отправку формы (конверсия).
4. Ссылки на сторы/сайты проектов уже проставлены (Google Play M1 Radio, vaadbot.com, moraltogether.com).

## Отладка

`index.html?noanim` — отключает reveal-анимации (удобно для скриншотов/тестов).
